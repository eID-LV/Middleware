Latvia-eid Middleware version 1.1.0

for Mac OS X 10.5 (Leopard), universal (32-bit i386 and ppc)

Latvia-eid Middleware provides a set of libraries and utilities to work with Latvia eID smart cards. Its main focus is to support cryptographic operations of these cards, and facilitate their use in security applications such as authentication, mail encryption and digital signatures.

Latvia-eid Middleware implements the PKCS#11 API so applications supporting this API (such as Mozilla Firefox and Thunderbird) can use it.
